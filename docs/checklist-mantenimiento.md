# Checklist de mantenimiento — Radio La Mina

## Cada semana

- Revisar que la web carga correctamente.
- Comprobar que el formulario de contacto envía correos.
- Revisar que la emisión en directo funciona.
- Revisar si hay actualizaciones de plugins o tema.
- Comprobar que no hay errores visibles en las páginas principales.

## Cada mes

- Hacer una copia de seguridad completa.
- Actualizar WordPress, plugins y tema.
- Revisar usuarios administradores.
- Revisar enlaces del menú y footer.
- Comprobar que HTTPS sigue activo.
- Revisar formularios y correo SMTP.

## Cada trimestre

- Revisar textos legales.
- Revisar rendimiento en móvil.
- Comprobar contenidos antiguos.
- Revisar si hay plugins no utilizados.
- Revisar copias de seguridad guardadas.

## Antes de actualizaciones importantes

- Hacer backup completo.
- Actualizar primero en entorno de pruebas si existe.
- Revisar web después de la actualización.
- Probar formulario y streaming.
