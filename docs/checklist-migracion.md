# Checklist de migración

- Hacer backup completo.
- Revisar web en entorno dev.
- Confirmar que todas las páginas están acabadas.
- Revisar menú y footer.
- Comprobar formulario.
- Comprobar SMTP.
- Comprobar streaming.
- Migrar archivos a producción.
- Actualizar URL de WordPress.
- Reemplazar URLs internas.
- Activar HTTPS.
- Forzar HTTPS.
- Revisar radiolamina.com.
- Revisar versión móvil.
- Desactivar “disuadir motores de búsqueda”.
- Hacer backup final de producción.
